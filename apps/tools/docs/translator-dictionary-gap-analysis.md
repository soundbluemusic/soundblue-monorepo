## 4. 테스트 단어 상세 분석

### 4.1 김밥 (Gimbap/Kimbap)

- **현재 상태**: 미등록
- **문제점**: 한국 대표 음식임에도 누락
- **필요 조치**: koToEnWords에 `김밥: 'gimbap'` 또는 `'kimbap'` 추가

### 4.2 피아노 (Piano)

- **현재 상태**: 미등록
- **문제점**: 악기 카테고리 전체가 누락됨
- **필요 조치**: 악기 카테고리 신설 및 주요 악기 30+ 추가

### 4.3 인형 (Doll)

- **현재 상태**: 미등록
- **문제점**: 장난감/완구 카테고리 전체가 누락됨
- **필요 조치**: 장난감 카테고리 신설 및 주요 항목 추가

### 4.4 공기청정기 (Air Purifier)

- **현재 상태**: 미등록
- **문제점**: 가전제품 카테고리 전체가 누락됨
- **복합어 분해 가능성**: `공기` + `청정` + `기` → 복합어 사전에도 없음
- **필요 조치**:
  - 가전제품 카테고리 신설
  - 또는 복합어 사전(compound-words.ts)에 추가

### 4.5 쇼핑몰 (Shopping Mall)

- **현재 상태**: 미등록 (`마트`, `백화점`만 있음)
- **문제점**: 현대적인 상업 시설 용어 부족
- **필요 조치**: 쇼핑/상업시설 관련 단어 확장

## 5. 근본 원인 분석

### 5.1 사전 설계 문제

1. **초기 설계 시 어휘 선정 기준 부재**
   - 체계적인 카테고리 분류 없이 임의 추가
   - 빈도수 기반 선정 미적용

2. **카테고리별 균형 미고려**
   - 특정 카테고리(대명사, 시간)는 비교적 완성
   - 다른 카테고리는 거의 비어있음

3. **현대 생활 반영 미흡**
   - IT/기술, 가전, 인터넷 관련 어휘 전무
   - 2020년대 일상용어 누락

### 5.2 복합어 처리 한계

현재 복합어 사전(`compound-words.ts`)에도 **54개**만 등록:
- 국적 관련: 7개
- 시간 관련: 5개
- 음식 관련: 3개
- 장소 관련: 6개

`공기청정기` 같은 복합어는:
- 단어 사전에도 없음
- 복합어 사전에도 없음
- 자동 분해도 불가능 (접미사 패턴에 `기`가 없음)

## 7. 결론

현재 번역기의 단어 사전은 **심각하게 부족**한 상태입니다.

- **현재**: ~450단어 (기본 회화의 9~15% 수준)
- **목표**: 최소 3,000단어 (기본 회화 가능 수준)
- **격차**: ~2,550단어 부족

사용자가 테스트한 5개 단어가 **모두 번역 불가**한 것은 이러한 근본적인 어휘 부족 때문입니다. 단순한 단어 번역도 안 되는 현 상태에서는 문장 번역의 정확도를 논하기 어렵습니다.

**즉각적인 어휘 확장 작업이 필수적**입니다.
