# 비유 표현 번역 규칙 (Figurative Expression Translation Rules)

> 한국어 ↔ 영어 번역 시 비유적 표현 변환 패턴

---

# 1. 직유 (Simile) 변환 규칙

## 비유사 매핑

| 한국어 비유사 | 영어 비유사 |
|---------------|-------------|
| ~처럼 | like ~ / as ~ |
| ~같이 | like ~ |
| ~같은 | like ~ / ~-like |
| ~듯이 | as if ~ / as though ~ |
| ~듯 | like ~ / as ~ |
| ~인 양 | as if ~ / as though ~ |
| ~만큼 | as ~ as |
| 마치 ~처럼 | just like ~ / as if ~ |
| 꼭 ~같은 | just like ~ / exactly like ~ |

## 변환 패턴

### 패턴 1: [명사]처럼 [형용사]
```
한: A는 B처럼 C하다
영: A is as C as B / A is C like B

예:
눈처럼 하얀 → white as snow
얼음처럼 차가운 → cold as ice
바람처럼 빠른 → fast as the wind
```

### 패턴 2: [명사]처럼 [동사]
```
한: A가 B처럼 C하다
영: A C like B

예:
새처럼 날다 → fly like a bird
물처럼 흐르다 → flow like water
아이처럼 울다 → cry like a child
```

### 패턴 3: [명사]같은 [명사]
```
한: B같은 A
영: A like B / B-like A

예:
천사같은 미소 → a smile like an angel's / an angelic smile
호랑이같은 남자 → a man like a tiger / a tiger-like man
꿀같은 목소리 → a voice like honey / a honeyed voice
```

### 패턴 4: 마치 [절]처럼
```
한: 마치 ~인 것처럼
영: as if ~ / as though ~

예:
마치 꿈인 것처럼 → as if it were a dream
마치 처음 보는 것처럼 → as though seeing it for the first time
```

# 2. 은유 (Metaphor) 변환 규칙

## 변환 패턴

### 패턴 1: A는 B이다
```
한: A는 B이다 / A는 B다
영: A is B / A is a B

예:
인생은 여행이다 → Life is a journey
시간은 금이다 → Time is gold
지식은 힘이다 → Knowledge is power
```

### 패턴 2: A는 B(이)요
```
한: A는 B(이)요 / A는 B로다
영: A is B

예:
내 마음은 호수요 → My heart is a lake
봄은 고양이로다 → Spring is a cat
```

### 패턴 3: B의 A (소유격 은유)
```
한: B의 A
영: A of B / B's A

예:
희망의 빛 → light of hope / hope's light
슬픔의 바다 → sea of sorrow / ocean of grief
시간의 강 → river of time
```

### 패턴 4: B인 A (수식 은유)
```
한: B인 A
영: A that is B / A, the B

예:
불꽃인 사랑 → love that is flame
감옥인 마음 → heart that is a prison
```

# 3. 의인법 (Personification) 변환 규칙

## 인간 동사 매핑

| 한국어 | 영어 |
|--------|------|
| 속삭이다 | whisper |
| 노래하다 | sing |
| 춤추다 | dance |
| 웃다 / 미소 짓다 | smile / laugh |
| 울다 | cry / weep |
| 말하다 | speak / say / tell |
| 손짓하다 | beckon / wave |
| 고개 숙이다 | bow |
| 기다리다 | wait |
| 잠들다 | sleep |
| 화내다 | rage / get angry |
| 부르다 | call |
| 두드리다 | knock |
| 어루만지다 | caress / stroke |

## 변환 패턴

### 패턴: [무생물 주어] + [인간 동사]
```
한: [무생물]이/가 [인간동사]하다
영: The [inanimate] [human verb]s

예:
바람이 속삭인다 → The wind whispers
달이 미소 짓는다 → The moon smiles
꽃이 고개를 숙인다 → Flowers bow their heads
파도가 해변을 어루만진다 → Waves caress the shore
시간이 기다려주지 않는다 → Time waits for no one
죽음이 문을 두드린다 → Death knocks on the door
```

# 4. 과장법 (Hyperbole) 변환 규칙

## 과장 표현 매핑

| 한국어 패턴 | 영어 패턴 |
|-------------|-----------|
| 죽을 만큼 ~ | ~ to death / dying of ~ |
| ~해 죽겠다 | dying to ~ / ~ to death |
| 백 번 / 천 번 / 만 번 | a hundred / thousand / million times |
| 눈이 빠지게 | until one's eyes fall out |
| 귀에 못이 박히게 | until it's drilled into one's ears |
| 배가 등에 붙다 | starving to death |
| 목이 빠지게 | craning one's neck |
| 산더미 같은 | a mountain of ~ |
| 바다 같은 (양) | an ocean of ~ |

## 변환 패턴
```
한: 죽을 만큼 아프다
영: It hurts to death / It hurts like hell

한: 백 번도 더 말했다
영: I've told you a million times

한: 눈물의 바다
영: An ocean of tears
```

# 6. 역설/모순어법 (Paradox/Oxymoron) 변환 규칙

## 변환 패턴

### 패턴 1: [형용사] + [반대 명사]
```
한: [형용사A] + [반대되는 명사B]
영: [Adjective A] + [Contradicting Noun B]

예:
차가운 불꽃 → cold fire
뜨거운 얼음 → hot ice
밝은 어둠 → bright darkness
달콤한 고통 → sweet pain
아름다운 상처 → beautiful scars
시끄러운 침묵 → loud silence
살아있는 죽음 → living death
잔인한 친절 → cruel kindness
```

### 패턴 2: [명사] + 없는/없이 + [관련 명사]
```
한: [명사A] 없는 [관련 명사B]
영: [Noun B] without [Noun A] / [Adjective]-less [Noun B]

예:
소리 없는 아우성 → silent scream / soundless cry
빛 없는 태양 → sunless light / light without sun
눈물 없는 슬픔 → tearless sorrow
```

### 패턴 3: 부정 + 긍정 결합
```
한: [긍정] 속의 [부정] / [부정] 속의 [긍정]
영: [Positive] in [Negative] / [Negative] in [Positive]

예:
절망 속의 희망 → hope in despair
어둠 속의 빛 → light in darkness
슬픔 속의 기쁨 → joy in sorrow
```

# 7. 환유 (Metonymy) 변환 규칙

## 유형별 변환 패턴

### 유형 1: 장소 → 기관/조직
```
한: [장소명]이/가 [동사]
영: [Place name] [verb]

예:
청와대가 발표했다 → The Blue House announced
백악관이 결정했다 → The White House decided
월가가 반응했다 → Wall Street reacted
할리우드가 만든 → Made by Hollywood
펜타곤이 확인했다 → The Pentagon confirmed
```

### 유형 2: 도구 → 행위/결과
```
한: [도구]로 [행위 암시]
영: The [tool] [implies action]

예:
펜은 칼보다 강하다 → The pen is mightier than the sword
(펜=글/언론, 칼=무력)

그는 펜을 놓았다 → He put down his pen
(펜=글쓰기/작가 활동)

칼을 들다 → take up the sword
(칼=전쟁/무력)
```

### 유형 3: 재료 → 제품
```
한: [재료]를 [동사]
영: [Material] [verb]

예:
유리를 깨다 → break the glass (유리=창문/거울)
철을 다루다 → work with steel (철=무기/공업)
```

### 유형 4: 작가 → 작품
```
한: [작가명]을/를 읽다/보다
영: read/watch [Author name]

예:
셰익스피어를 읽다 → read Shakespeare
피카소를 보다 → see Picasso
베토벤을 듣다 → listen to Beethoven
```

### 유형 5: 상징물 → 본체
```
한: [상징물]을/를 [동사]
영: [Symbol] [verb]

예:
왕관을 쓰다 → wear the crown (왕관=왕위)
십자가를 지다 → bear the cross (십자가=고난/희생)
칼을 꺾다 → sheathe the sword (칼=전쟁)
```

# 8. 제유 (Synecdoche) 변환 규칙

## 유형별 변환 패턴

### 유형 1: 부분 → 전체
```
한: [신체 부분]이/가 [동사]
영: [Body part] [verb] / need more [body part]s

예:
손이 모자라다 → We need more hands (손=일꾼)
머리를 세다 → count heads (머리=사람)
입이 많다 → too many mouths to feed (입=사람)
지붕 아래 살다 → live under one roof (지붕=집)
```

### 유형 2: 전체 → 부분
```
한: [전체]가 [동사]
영: [Whole] [verb]

예:
온 세상이 알았다 → The whole world knew (세상=사람들)
한국이 울었다 → Korea wept (한국=한국인들)
```

### 유형 3: 종(種) → 류(類)
```
한: [구체적 명사]로 [일반] 지칭
영: [Specific noun] for [general]

예:
빵을 달라 → Give us bread (빵=식량)
밥을 먹다 → have rice (밥=식사)
```

### 유형 4: 류(類) → 종(種)
```
한: [일반 명사]로 [구체적] 지칭
영: [General noun] for [specific]

예:
그 생물은 위험하다 → That creature is dangerous (생물=특정 동물/인간)
```

## 한영 관용적 제유 대응

| 한국어 | 영어 | 의미 |
|--------|------|------|
| 손이 모자라다 | need more hands | 일꾼 부족 |
| 머리 수를 세다 | count heads | 인원 파악 |
| 입을 놀리다 | run one's mouth | 말하다 |
| 발을 끊다 | cut ties | 관계 단절 |
| 눈을 주다 | give the eye | 관심 보이다 |
| 귀를 기울이다 | lend an ear | 경청하다 |
| 지붕 아래 | under one roof | 한 집에 |
| 백의민족 | white-clad nation | 한민족 |

# 11. 번역 알고리즘 규칙 요약

## 비유 감지 패턴 (한국어)
```
직유: ~처럼, ~같이, ~같은, ~듯이, ~듯, ~인 양, ~만큼
은유: A는 B이다, A는 B다, A는 B(이)요, A는 B로다
의인법: [무생물 주어] + [인간 동사]
과장: 죽을 만큼, ~해 죽겠다, 백/천/만 번
역설: [형용사] + [반대 명사], [명사] 없는 [명사]
환유: [장소/도구/재료/작가/상징] → [기관/행위/제품/작품/본체]
제유: [부분] → [전체], [전체] → [부분]
```

## 비유 감지 패턴 (영어)
```
Simile: like ~, as ~ as, as if ~, as though ~
Metaphor: A is B, A is a B, A of B
Personification: [inanimate subject] + [human verb]
Hyperbole: to death, dying of, million times
Paradox/Oxymoron: [adjective] + [contradicting noun]
Metonymy: [place/tool/material/author/symbol] → [institution/action/product/work/entity]
Synecdoche: [part] → [whole], [whole] → [part]
```

## 변환 우선순위
1. 관용적 비유 대응표 확인 (1:1 매핑)
2. 문화 특수 비유 확인 (의역 필요)
3. 패턴 기반 변환 적용
4. 직역 (패턴 미감지 시)
