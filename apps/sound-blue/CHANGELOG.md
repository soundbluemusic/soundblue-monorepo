## [3.0.18-베타] - 2025-12-15

### Added

- Chat system with NLP-based topic detection
- Bilingual keyword detection (English/Korean)
- Comprehensive test coverage (26 test files)

### Changed

- Improved documentation accuracy
- Added Chat section to CLAUDE.md and README.md

## [3.0.17-베타] - 2025-12-08

### Changed

- Updated documentation structure
- Improved i18n translations

---

## [3.0.10-베타] - 2025-12-08

### Added

- Sitemap structure template documentation in CLAUDE.md, README.md, next-sitemap.config.js
- X-DEFAULT and LASTMOD fields to sitemap structure
- Version badges to README.md

### Changed

- Improved README.md for human readers (simplified structure, removed verbose prompts)
- Improved CLAUDE.md for AI assistants (streamlined Biome config, updated scripts)
- Updated documentation to reflect current project state

## [3.0.9-베타] - 2025-12-07

### Added

- GitHub contributors mapping (.mailmap)

---

## [3.0.0-베타] - 2025-12-01

### Added

- React 19.1 with React Router 7.6 (SSG mode)
- TypeScript 5.9 strict mode
- Tailwind CSS 4.x styling system
- Pure CSS Particles (lightweight visual effects)
- Custom i18n with Zustand (Korean/English)
- vite-plugin-pwa PWA support
- Biome 2.x linting and formatting
- Vitest unit testing
- Playwright E2E testing
- Cloudflare Pages static deployment

### Changed

- New design system with CSS variables (Eye-Friendly theme)
- Improved accessibility features (WCAG compliance)
