## 통계

- **총 표제어**: ~500개
- **총 의미 항목**: ~1,200개
- **평균 의미 수**: 2.4개/단어

## 사용 예시

```javascript
// 문맥 기반 의미 선택
function selectSense(word, context) {
  const entry = dictionary[word];
  for (const sense of entry.senses) {
    if (sense.context.some(c => context.includes(c))) {
      return sense;
    }
  }
  return entry.senses[0]; // 기본값
}

// 예: "bank" + "river" → bank.2 (강둑)
// 예: "bank" + "account" → bank.1 (은행)
```
