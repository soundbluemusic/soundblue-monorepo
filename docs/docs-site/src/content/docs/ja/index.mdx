## About

**Sound Blue**は韓国のインディーアーティストです。

**SoundBlueMusic**はすべてのプロジェクトが集まる場所です — 音楽、ツール、アプリ。

- **無料** — 登録不要、広告なし
- **オープンソース** — GitHubでコード公開
- **多言語** — EN / KO / JA
