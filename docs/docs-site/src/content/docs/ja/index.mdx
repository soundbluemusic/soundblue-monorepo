## SoundBlueとは？

SoundBlueは**SoundBlueMusic**のクリエイティブプラットフォームです。音楽を作り、クリエイターのためのツールを開発しています。

すべてが：

- **無料** — 登録不要、広告なし、有料機能なし
- **多言語対応** — 英語と韓国語をサポート
- **オフライン対応** — インターネットなしで動作（PWA）
- **オープンソース** — コードはGitHubで公開
