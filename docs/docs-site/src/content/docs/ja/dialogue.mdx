**100%オフラインで動作するQ&Aツール。**

インターネット接続なしで即座に回答。

**ウェブサイト**: [dialogue.soundbluemusic.com](https://dialogue.soundbluemusic.com)

---

## 主な機能

| 機能 | 説明 |
|------|------|
| 🔌 **オフライン** | 完全にインターネットなしで動作 |
| ⚡ **即座** | 遅延なしの即時回答 |
| 🌏 **バイリンガル** | 英語と韓国語をサポート |
| 📱 **PWA** | アプリとしてインストール |
| ♿ **アクセシブル** | WCAG準拠、キーボードナビゲーション |

---

## 仕組み

Dialogueは一般的なQ&Aアプリとは異なります：

1. **すべてのデータが組み込み** — サーバー呼び出し不要
2. **オフラインで動作** — 一度ロードすれば、インターネットはオプション
3. **プライバシー優先** — デバイスを離れるデータなし

---

## 使用例

- インターネットなしでクイックリファレンス
- 言語練習のための学習ツール
- オフラインドキュメントアクセス

## アプリとしてインストール

DialogueはプログレッシブWebアプリ（PWA）です：

1. デバイスでウェブサイトにアクセス
2. 「インストール」または「ホーム画面に追加」を探す
3. ネイティブアプリのように使用 — オフラインでも
