| 機能 | 説明 |
|------|------|

---

## 仕組み

Dialogueは一般的なQ&Aアプリとは異なります：

1. **すべてのデータが内蔵** — サーバー呼び出し不要
2. **オフライン動作** — 一度ロードすれば、インターネットは任意
3. **プライバシー優先** — デバイスの外には何も送信されない

## アプリとしてインストール

DialogueはProgressive Web App（PWA）です：

1. デバイスでウェブサイトにアクセス
2. 「インストール」または「ホーム画面に追加」を選択
3. ネイティブアプリのように使用 — オフラインでも
