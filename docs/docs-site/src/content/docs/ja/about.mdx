## SoundBlueMusic

> *"こんにちは、sound blueです、音楽を作っています。"*

でもそれだけではありません。

Sound Blueは韓国のインディーアーティストです。SoundBlueMusicはすべてのクリエイティブプロジェクトが集まる場所です。

## 哲学

### 誰でも無料

すべてのツールとアプリは完全に無料です：

- 登録不要
- 広告なし
- 有料機能なし
- データ収集なし

### アクセシビリティ

私たちが作るすべてのものは：

- 複数の言語で利用可能（EN/KO）
- 障害のあるユーザーもアクセス可能（WCAG）
- 可能な限りオフラインで動作（PWA）

### オープンソース

コードは公開されていますが、コンテンツは著作権で保護されています。
