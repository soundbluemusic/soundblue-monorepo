## SoundBlueMusic

> _"hi, im sound blue, i make music."_

でもそれだけではありません。

**Sound Blue**は韓国のインディーアーティストです。**SoundBlueMusic**はすべてのクリエイティブプロジェクトが集まる場所です。

---

## 活動内容

| 分野 | 説明 |
|------|------|
| 🎵 **音楽** | オリジナル曲、アルバム、プロダクション |
| 🛠️ **ウェブツール** | ミュージシャンとクリエイターのための無料ツール |
| 💬 **アプリ** | オフライン優先、プライバシー重視のアプリケーション |
| 🌐 **オープンソース** | GitHubでコードを公開 |

---

## 哲学

### 誰でも無料

すべてのツールとアプリは完全に無料です：
- 登録不要
- 広告なし
- 有料機能なし
- データ収集なし

### アクセシビリティ

私たちが作るすべてのものは：
- 多言語対応（英語/韓国語）
- 障害者アクセシビリティ対応（WCAG）
- 可能な限りオフラインで動作（PWA）

### オープンソース

コードは公開されていますが、コンテンツには著作権があります。

## 著作権

SoundBlueMusicが作成したすべてのコンテンツには著作権があります。

ソースコードはオープンソースでGitHubで利用可能ですが：

これには音楽、アートワーク、テキスト、その他すべてのクリエイティブコンテンツが含まれます。
