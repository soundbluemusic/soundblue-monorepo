**모든 크리에이터를 위한 무료 웹 도구 플랫폼.**

뮤지션, 디자이너, 개발자를 위한 전문 온라인 도구.

**웹사이트**: [tools.soundbluemusic.com](https://tools.soundbluemusic.com)

---

## 핵심 가치

| 가치 | 설명 |
|------|------|
| 🆓 **무료** | 완전히 무료로 사용 |
| 🚫 **가입 불필요** | 계정 불필요 |
| 🚫 **광고 없음** | 광고 없음 |
| 🔒 **프라이버시** | 데이터는 기기에만 저장 |

---

## 사용 가능한 도구

### 음악 도구 🎵

| 도구 | 설명 |
|------|------|
| **메트로놈** | BPM 조절, 박자, 펜듈럼 시각화 |
| **드럼 머신** | 다양한 사운드가 있는 드럼 패턴 시퀀서 |
| **탭 템포** | 탭하여 BPM 감지 |
| **딜레이 계산기** | BPM에서 딜레이 시간 계산 |

### 언어 도구 🌐

| 도구 | 설명 |
|------|------|
| **번역기** | 한영 양방향 번역 |
| **맞춤법 검사기** | 한국어 맞춤법 검사 |
| **영어 맞춤법 검사기** | 영어 맞춤법 검사 |

### 시각 도구 🎨

| 도구 | 설명 |
|------|------|
| **QR 생성기** | QR 코드 즉시 생성 |
| **컬러 하모니** | 색상환 및 하모니 생성기 |
| **컬러 팔레트** | 커스텀 색상 조합 |
| **컬러 분해** | 색상 구성 요소 분석 |

---

## 특징

- **이중 언어** — 영어와 한국어 인터페이스
- **PWA** — 앱으로 설치 가능
- **오프라인** — 인터넷 없이 작동
- **반응형** — 모든 기기에서 작동

---

## 사용 방법

1. tools.soundbluemusic.com 방문
2. 사이드바에서 도구 선택
3. 바로 사용 시작 — 설정 불필요

모든 것이 브라우저에서 실행됩니다. 서버, 계정, 추적 없음.
