## 제공 도구

### 음악 도구 🎵

| 도구 | 설명 |
|------|------|
| **메트로놈** | BPM 조절, 박자 설정, 펜듈럼 시각화 |
| **드럼 머신** | 다양한 사운드의 드럼 패턴 시퀀서 |

### 유틸리티 도구 🛠️

| 도구 | 설명 |
|------|------|
| **QR 생성기** | 즉시 QR 코드 생성 |
| **번역기** | 한국어 ↔ 영어 양방향 번역 |

### 비주얼 도구 🎨

| 도구 | 설명 |
|------|------|
| **컬러 하모니** | 색상환과 하모니 생성기 |
| **컬러 팔레트** | 커스텀 색상 조합 |

## 특징

- **이중언어** — 영어와 한국어 인터페이스
- **PWA** — 앱으로 설치 가능
- **오프라인** — 인터넷 없이 작동
- **반응형** — 모든 기기에서 사용 가능
