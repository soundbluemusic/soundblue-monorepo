## SoundBlue란?

SoundBlue는 **SoundBlueMusic**의 크리에이티브 플랫폼입니다. 음악을 만들고, 창작자를 위한 도구를 개발합니다.

모든 것이:

- **무료** — 회원가입 없음, 광고 없음, 유료 기능 없음
- **이중언어** — 영어와 한국어 지원
- **오프라인 가능** — 인터넷 없이 작동 (PWA)
- **오픈소스** — 코드는 GitHub에 공개
