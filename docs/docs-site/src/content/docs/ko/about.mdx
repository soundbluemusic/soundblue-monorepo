## SoundBlueMusic

> _"hi, im sound blue, i make music."_

하지만 그게 전부가 아닙니다.

**SoundBlueMusic**은 한국의 인디 아티스트이자 크리에이터입니다. 음악이 모든 것의 중심이지만, 창작 활동은 노래를 넘어 확장됩니다.

---

## 하는 일

| 분야 | 설명 |
|------|------|
| 🎵 **음악** | 오리지널 곡, 앨범, 프로덕션 |
| 🛠️ **웹 도구** | 뮤지션과 창작자를 위한 무료 도구 |
| 💬 **앱** | 오프라인 우선, 프라이버시 중심 애플리케이션 |
| 🌐 **오픈소스** | GitHub에 코드 공개 |

---

## 철학

### 모두에게 무료

모든 도구와 앱은 완전히 무료입니다:
- 회원가입 불필요
- 광고 없음
- 유료 기능 없음
- 데이터 수집 없음

### 접근성

우리가 만드는 모든 것은:
- 다국어 지원 (영어/한국어)
- 장애인 접근성 (WCAG)
- 가능하면 오프라인 작동 (PWA)

### 오픈소스

코드는 공개되지만, 콘텐츠는 저작권이 있습니다.

## 저작권

SoundBlueMusic이 만든 모든 콘텐츠는 저작권이 있습니다.

소스 코드는 오픈소스로 GitHub에서 이용 가능하지만:

- **코드**: 오픈소스 (MIT 라이선스)
- **콘텐츠**: 모든 권리 보유

여기에는 음악, 아트워크, 텍스트, 기타 모든 창작 콘텐츠가 포함됩니다.
