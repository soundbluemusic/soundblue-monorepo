## SoundBlueMusic

> *"안녕, 나는 sound blue, 음악을 만들어."*

하지만 그게 전부가 아닙니다.

Sound Blue는 한국의 인디 아티스트입니다. SoundBlueMusic은 모든 창작 프로젝트가 모여있는 곳입니다.

---

## 하는 일

| 분야 | 설명 |
|------|------|
| 🎵 **음악** | 오리지널 곡, 앨범, 프로덕션 |
| 🛠️ **웹 도구** | 뮤지션과 크리에이터를 위한 무료 도구 |
| 💬 **앱** | 오프라인 우선, 프라이버시 중심 앱 |
| 🌐 **오픈 소스** | GitHub에 공개된 코드 |

---

## 철학

### 모두에게 무료

모든 도구와 앱은 완전히 무료입니다:

- 가입 불필요
- 광고 없음
- 유료 기능 없음
- 데이터 수집 없음

### 접근성

우리가 만드는 모든 것은:

- 여러 언어로 이용 가능 (EN/KO)
- 장애가 있는 사용자도 접근 가능 (WCAG)
- 가능한 오프라인 작동 (PWA)

### 오픈 소스

코드는 공개되어 있지만, 콘텐츠는 저작권이 있습니다.
