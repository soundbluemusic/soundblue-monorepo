## 작동 원리

Dialogue는 일반적인 Q&A 앱과 다릅니다:

1. **모든 데이터가 내장됨** — 서버 호출 불필요
2. **오프라인 작동** — 한 번 로드하면 인터넷 선택 사항
3. **프라이버시 우선** — 기기 밖으로 아무것도 나가지 않음

## 앱으로 설치

Dialogue는 프로그레시브 웹 앱(PWA)입니다:
