**100% 오프라인으로 작동하는 Q&A 도구.**

인터넷 연결 없이 즉시 답변.

**웹사이트**: [dialogue.soundbluemusic.com](https://dialogue.soundbluemusic.com)

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| 🔌 **오프라인** | 완전히 인터넷 없이 작동 |
| ⚡ **즉시** | 지연 없는 즉시 답변 |
| 🌏 **이중 언어** | 영어와 한국어 지원 |
| 📱 **PWA** | 앱으로 설치 가능 |
| ♿ **접근성** | WCAG 준수, 키보드 내비게이션 |

---

## 작동 방식

Dialogue는 일반적인 Q&A 앱과 다릅니다:

1. **모든 데이터가 내장** — 서버 호출 불필요
2. **오프라인 작동** — 한번 로드되면 인터넷 선택
3. **프라이버시 우선** — 기기를 떠나는 데이터 없음

## 앱으로 설치

Dialogue는 프로그레시브 웹 앱(PWA)입니다:

1. 기기에서 웹사이트 방문
2. "설치" 또는 "홈 화면에 추가" 찾기
3. 네이티브 앱처럼 사용 — 오프라인에서도
