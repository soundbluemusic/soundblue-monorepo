## Philosophy

### Free for Everyone

All tools and apps are completely free:

- No sign-up required
- No advertisements
- No paywalls
- No data collection

### Accessible

Everything we build is:

- Available in multiple languages (EN/KO)
- Accessible to users with disabilities (WCAG)
- Works offline when possible (PWA)

### Open Source

The code is public, but the content is copyrighted.
