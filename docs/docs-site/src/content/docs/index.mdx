## What is SoundBlue?

SoundBlue is a creative platform by **SoundBlueMusic** — an indie artist who makes music and builds tools for creators.

Everything here is:

- **Free** — No sign-up, no ads, no paywalls
- **Bilingual** — English and Korean supported
- **Offline-capable** — Works without internet (PWA)
- **Open source** — Code is public on GitHub
